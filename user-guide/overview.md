# User Guide Overview

The TTS Communication Tool is designed around a simple, fast workflow. This guide covers every user-facing feature.

## Core Workflow

```mermaid
graph LR
    A[Press Hotkey] --> B[Overlay Appears]
    B --> C[Type Message]
    C --> D[Press Enter]
    D --> E[TTS Generates Audio]
    E --> F[Play to Monitor]
    E --> G[Play to Virtual Cable]
    F --> H[You Hear It]
    G --> I[Voice App Transmits It]
```
