# Keyboard Shortcuts

## Global Hotkeys

These hotkeys work from anywhere in Windows, even when other applications are focused.

| Action | Default | Configurable |
|--------|---------|-------------|
| Open overlay | Ctrl+Shift+Space | Yes |
| Stop playback | Ctrl+Shift+Backspace | Yes |
| Open Settings | (none) | Yes |
| Resend last message | (none) | Yes |

## Overlay Hotkeys

These hotkeys work while the overlay window is open.

| Action | Key | Notes |
|--------|-----|-------|
| Send text | Enter | Triggers TTS and playback |
| Stop + Send | Ctrl+Enter | Stops current audio, then sends overlay text |
| Close overlay | Esc | Closes without sending |
| Close overlay | Click outside | Closes without sending |
| Close overlay | X button | Closes without sending |

## Settings Window

| Action | Key |
|--------|-----|
| Save | Ctrl+S or Save button |
| Switch tabs | Click tab headers |

## Phrase Manager

| Action | Key |
|--------|-----|
| Add phrase | Add button |
| Delete phrase | Delete button (with confirmation) |
| Play phrase | Play button |
| Edit phrase | Select from list, edit fields |

## Phrase Editor

| Action | Key |
|--------|-----|

| Delete phrase | Delete button (with confirmation callback) |

## Tray Icon

|--------|--------|
